# Name Use Policy

The code of JS8Mesh is licensed under `GPL-3.0-only`, but the project name is
meant to identify the original project by its original author.

If you create and distribute a modified version or a fork:

- please use a different name
- please do not present the fork as the original JS8Mesh
- please make it clear to users that your version is modified

This is requested to avoid confusion for operators, testers, and users of the
original project.

Suggested wording for forks:

- "Based on JS8Mesh"
- "Forked from JS8Mesh"
- "Modified from JS8Mesh"
